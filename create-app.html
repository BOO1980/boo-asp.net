Create an ASP.NET app

to create an app, in your command prompt use the following command

dotnet new webapp -o -MyWebApp --no-https -f net7.0

What does this comman mean?

The dotnet new command creates a new application

  • The webApp parameter selects what template to use when creating your app
  • The -o parameter create a directory named MyWebApp where your app is stored
  • The --no-https flag specifies not to enable HTTPS
  • The -f parameter indiciates you're creating a .NET 7 application
